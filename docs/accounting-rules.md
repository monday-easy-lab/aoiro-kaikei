## 2. 開業前に支払った費用（開業費）

### 開業費とは

開業準備のために支払った費用は、**開業日以降に経費として計上**できます。本アプリでは「かんたん入力」で通常の経費と同じように入力し、メモに購入日を記録します。

### 入力のルール

| 購入品 | 金額 | 入力方法 | 勘定科目 |
|---|---|---|---|
| サブスク（Adobe, ChatGPT等） | 金額問わず | 経費 → ソフトウェア利用料 | ソフトウェア利用料 |
| サーバー・ドメイン代 | 金額問わず | 経費 → 通信費 | 通信費 |
| 名刺・チラシ作成 | 金額問わず | 経費 → 広告宣伝費 | 広告宣伝費 |
| 書籍・教材 | 金額問わず | 経費 → 書籍・資料を買った | 新聞図書費 |
| PC・カメラ・機材（10万円以上） | 10万円以上 | 固定資産 → 備品を買った | 工具器具備品 |
| PC・カメラ・機材（10万円未満） | 10万円未満 | 経費 → 消耗品を買った | 消耗品費 |
| iPhone（事業使用分のみ） | 按分後の金額 | 経費 → 消耗品を買った | 消耗品費 |

### 共通のポイント

- **日付:** 開業日（例: 2026-02-01）で入力する
- **支払方法:** 開業前は事業口座がないので **🙋 個人立替** を選択
- **メモ:** 「開業準備費（実際の購入日: 2025/12/10）」と実際の購入日を記録

### 入力例: 開業前にAdobeを契約していた

```
カテゴリ:   🛒 経費
テンプレート: ソフトウェア利用料を払った
日付:       2026-02-01（開業日）
金額:       6,480円
支払方法:   🙋 個人立替
メモ:       開業準備費 Adobe CC（2025/12/10契約）

→ 仕訳: ソフトウェア利用料 / 事業主借
```

### 入力例: 開業前にPCを買った（15万円）

```
カテゴリ:   🖥️ 固定資産
テンプレート: 10万円以上の備品を買った
日付:       2026-02-01（開業日）
金額:       150,000円
支払方法:   🙋 個人立替
メモ:       開業準備費 MacBook Air（2025/11/20購入）

→ 仕訳: 工具器具備品 / 事業主借
```

> ⚠ **注意:** 開業費の計上方法は複数あり（繰延資産として任意償却等）、ここで紹介しているのは最もシンプルな方法です。金額が大きい場合や判断に迷う場合は、税理士にご相談ください。

## 3. クレジットカードの仕訳フロー

クレカは「使った日」と「引き落とされる日」が異なるため、2段階で記録します。

```
┌─────────────────────┐
│ ① 経費発生日        │
│ 例: 6/15にAdobe決済  │
│                     │
│ ソフトウェア利用料    │
│    / クレジットカード │
│                     │
│ → クレカ残高が増える │
└────────┬────────────┘
         │
         ▼
┌─────────────────────┐
│ ② 引落日            │
│ 例: 7/27に口座引落   │
│                     │
│ クレジットカード      │
│    / 普通預金        │
│                     │
│ → クレカ残高が消える │
│ → 預金が減る         │
└─────────────────────┘
```

### よくある間違い

| ❌ やりがちなミス | ✅ 正しい方法 |
|---|---|
| 引落日に経費を入力 | 経費は使った日に入力。引落日は「資金移動→クレカの引き落とし」 |
| 経費と引落の両方で経費計上 | 経費は1回だけ。引落はクレカ残高の決済処理 |
| 個人カードの利用をクレカで入力 | 個人カードは「🙋 個人立替」で入力。引落処理は不要 |

## 4. 個人立替と事業主借

### いつ使う？

個人の財布・個人カード・個人の銀行口座で事業の支払いをした場合。

### 具体例

- 個人のクレジットカードでAdobe CC を支払った
- 個人の現金で打ち合わせのカフェ代を払った
- 個人の口座からサーバー代が引き落とされた

### 入力方法

かんたん入力で経費テンプレートを選び、支払方法で **🙋 個人立替** を選ぶだけ。裏では「事業主借」として処理されます。

### 個人口座と事業口座が同じ場合

口座を分けていない場合は **🏦 銀行振込** を選んでください。事業口座から直接支払った扱いになります。

### 年度末の処理

事業主借は年度締め時に元入金に自動振替されます。追加の操作は不要です。

## 5. 家事按分の考え方

### 按分とは

自宅兼事務所の家賃や、私用でも使うスマホ代など、事業とプライベートの両方にかかる費用を「事業使用割合」で分けることです。

### 按分の目安

| 科目 | 計算方法の例 | 目安 |
|---|---|---|
| 家賃 | 仕事部屋の面積 ÷ 家全体の面積 | 20〜50% |
| 電気代 | 仕事時間 ÷ 24時間 | 25〜40% |
| 通信費（Wi-Fi） | 仕事での利用割合 | 30〜50% |
| スマホ代 | 仕事での利用割合 | 30〜50% |
| 車両費 | 仕事の走行距離 ÷ 総走行距離 | 状況次第 |

### 本アプリでの設定方法

1. **設定 → 家事按分** でスライダーを調整
2. 日常の入力は**全額で入力**（按分計算は不要）
3. 損益計算書と所得税計算に**自動的に按分後の金額が反映**される

### 個別の購入品（PC・タブレット等）

按分設定は科目単位のため、個別の購入品は**事業使用分の金額だけを入力**してください。

```
例: タブレット 50,000円（事業使用70%）
→ 消耗品費 35,000円 で入力
→ メモ: 「タブレット 50,000円×事業70%」
```

## 6. 減価償却の基本

### いつ必要？

10万円以上の事業用資産（PC、カメラ、車など）を購入した場合、購入年に全額を経費にできず、数年に分けて経費にします。これが減価償却です。

### 主な耐用年数

| 資産 | 耐用年数 | 年間の償却額（定額法） |
|---|---|---|
| パソコン | 4年 | 取得価額 ÷ 4 |
| カメラ | 5年 | 取得価額 ÷ 5 |
| 自動車（普通車） | 6年 | 取得価額 ÷ 6 |
| 自動車（軽自動車） | 4年 | 取得価額 ÷ 4 |

> ※ 定額法の場合。個人事業主のデフォルトは定額法です。

### 本アプリでの入力

1. 購入時: **固定資産 → 備品を買った**（取得価額で入力）
2. 年末: **固定資産 → 減価償却を計上した**（年間償却額を入力）

### 10万円未満の場合

10万円未満の場合は固定資産ではなく **消耗品費** で全額をその年の経費にできます。

### 注意

本アプリには減価償却の自動計算機能はありません。耐用年数と償却額は手動で計算してください。国税庁の「減価償却資産の耐用年数表」を参照するか、税理士にご相談ください。
